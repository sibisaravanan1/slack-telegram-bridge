# Slack-Telegram-Bridge

A bridge between slack and telegram channels.


## Tech Stack

**Server:** Python, Flask


## Use case

- Message sent on slack channel will be forwarded to Telegram channel.
# Python :

- PyTelegramBotAPI (https://github.com/eternnoir/pyTelegramBotAPI)
- slackeventsapi
- slack_sdk
- requests
